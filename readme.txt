== Description ==

Shows basic user data from Filmweb.pl portal.
For performance reasons the data is refreshed and parsed once every 6 hours.

== Installation ==

1. Download *.zip file with plugin contents
2. Extract plugin from the *.zip file
3. Upload all extracted files and folders to the `/wp-content/plugins/wp-filmweb-widget` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Enable and configure the widget in the Widgets menu
